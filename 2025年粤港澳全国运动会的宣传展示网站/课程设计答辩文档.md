## 一、项目概述

### 1.1 项目背景

本项目是为2025年第十五届全国运动会（粤港澳赛区）开发的官方宣传展示网站。粤港澳三地首次联合举办全运会，开创了跨境办赛的先河，具有历史性意义。本网站旨在全面展示赛事信息、运动员风采、新闻动态等内容，为公众提供便捷的信息获取渠道。

### 1.2 项目目标

- 建设一个功能完善、界面美观的运动会宣传网站
- 实现多页面联动与统一视觉风格
- 提供良好的用户体验（响应式设计、交互动画）
- 展示现代Web开发技术（CSS3动画、JavaScript高级应用）

### 1.3 项目结构

## 二、技术栈分析

### 2.1 前端技术架构

| 技术类别 | 具体技术 | 应用场景 |
|---------|---------|---------|
| **HTML5** | 语义化标签、data属性、ARIA无障碍 | 页面结构、数据存储、屏幕阅读器支持 |
| **CSS3** | 变量、Flexbox、Grid、动画、过渡 | 布局、样式、交互效果 |
| **JavaScript (ES6+)** | 类、Promise、async/await、Fetch API | 交互逻辑、数据请求、动画控制 |
| **Web API** | IntersectionObserver、Geolocation、Web Speech API | 性能优化、定位服务、语音识别 |

### 2.2 核心技术亮点

#### （1）纯CSS实现的筛选功能
网站采用**CSS复选框技术（Checkbox Hack）**实现分类筛选，无需JavaScript即可完成过滤逻辑。

```css
/* 核心原理：隐藏radio按钮，通过label控制显示/隐藏 */
.filter-radio { display: none; }

#filter-track:checked ~ .athletes-grid .athlete-card[data-category="track"] {
    display: flex;
}
```

#### （2）IntersectionObserver性能优化
使用现代API实现视口检测，避免频繁的scroll事件监听：

```javascript
// 图片懒加载
var obs = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            img.src = img.dataset.src;
            obs.unobserve(img);
        }
    });
});
```

#### （3）模块化JavaScript架构
采用**类（Class）**进行功能封装，实现关注点分离：

| 类名 | 功能 | 代码行数 |
|-----|------|---------|
| `GamesWebsite` | 核心功能（主题、轮播、搜索、表单） | 423行 |
| `AdvInteract` | 高级交互（标签页、拖拽、手势、键盘） | 361行 |
| `ExtraFunc` | 扩展功能（AJAX、灯箱、语音搜索） | 607行 |

checkField(f) {
    var v = f.value.trim();
    // 邮箱格式验证
    if (t === 'email' && v) {

#### 触摸手势识别：
```javascript
gesture() {
    var sx, sy, st;
    el.addEventListener('touchstart', (e) => {
        sx = e.touches[0].clientX;
        sy = e.touches[0].clientY;
        st = Date.now();
    });
    el.addEventListener('touchend', (e) => {
        var dx = sx - e.changedTouches[0].clientX;
        var dy = sy - e.changedTouches[0].clientY;
        var dt = Date.now() - st;
        if (Math.abs(dx) > 50 && dt < 300) {
            this.swipe(el, dx > 0 ? 'left' : 'right');
        }
    });
}
```
